Payment Entry, Account Entry

The user has two options to specify the Account for proper payment application.

  • Simply Type Account: If the user knows the exact account number, they can simply type the account. As the user's cursor leaves this field, the system will attempt to lookup the account name. If found, then the account name will fill the Name field.
  • Double Click the Account Text Box: The user will be presented with a lookup window.

Here is an example payment entry

The lower right area of the payment entry has an Account text input field.


When the user double clicks the Account field, they will be presented with another window like the following

The user enters a search term in the upper left area, and the system will populate the list of accounts that match the term. In this case the user was searching for the term "jacki", and the system found 3 accounts.

Here is another example where the user searched for "140".

After the user clicks or selects one of the accounts, the system will attempt to find any outstanding charges for the selected account. In this case the system found some outstanding/unpaid charges.
